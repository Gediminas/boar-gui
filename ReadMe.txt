boar-gui - Interface for Boar version control and backup

NOTES:
    * Boar - http://code.google.com/p/boar/
    * Tested (developed) on Windows 7 x64
    * Currently supports only these commands: boar info|log|status|commit
    * bin/bg.exe should be added to Windows environment variable "path"
      and launched from repository dir

Compile:
    Python   2.7.5    (win64)
    wxPython 2.8.12.1 (win64, unicode, python 2.7)
    py2exe   0.6.9    (win64, python 2.7)
